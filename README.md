# Customer Segmentation using K-Means

## Overview

This project applies unsupervised machine learning to segment customers based on their income and spending behavior.

## Objective

Identify distinct customer groups to support targeted marketing and business strategies.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib / Seaborn

## Workflow

* Data exploration
* Feature selection
* Elbow Method to find optimal clusters
* K-Means clustering
* Visualization of results

## Results

* Identified 5 distinct customer segments
* Clear separation between high-value and low-value customers

## Key Insights

* High-income, high-spending customers are the most valuable
* Some high-income customers spend less → opportunity for growth
* Segmentation enables targeted marketing strategies
